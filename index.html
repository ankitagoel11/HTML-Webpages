Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled the data from Open Weather API to assemble a dataset on over 500 countries.
After assembling the dataset, we used matplotlib library of python to plot various aspecs of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This also provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
